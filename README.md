# CubeArena

Aplicacion web para speedcubers que permite grabar miniclips de resoluciones, guardar tiempos y construir una base de datos de jugadores.

## Caracteristicas

- Grabacion de videos directamente desde la camara web
- Sistema de login con multiples opciones
- Guardado automatico de videos con nombre y tiempo
- Diseno responsivo

## Tecnologias utilizadas

- Frontend: HTML5, CSS3, JavaScript
- Backend: Node.js + Express
- Grabacion de video: WebRTC + MediaRecorder API

## Proximas mejoras

- Autenticacion real con Google
- Base de datos SQLite para historial de partidas
- Pagina de historial con todos los videos del usuario
- Eliminacion automatica de videos antiguos
- Graficos de progresion de tiempos
